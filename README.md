# Docker-Compose-Termius-AWS

# POR GIT CLONE

* CLONAR REPOSITÓRIO DESEJADO
  
```
git clone https://github.com/birazn/compose-exemplo.git
```

* MUDAR PARA O DIRETÓRIO CLONADO

```
cd compose-exemplo
```

* SUBIR ARQUIVOS DO COMPOSE
  
```
docker compose up -d
```
